# Calculator Project

This is a fully functional web-based calculator built as part of **The Odin Project** curriculum. It demonstrates foundational JavaScript skills, DOM manipulation, and event handling.

## Features

*   **Core Math Operations:** Supports addition, subtraction, multiplication, and division.
*   **Sequential Calculations:** Allows chaining multiple operations without clearing.
*   **Error Handling:** Displays a sarcastic error message when trying to divide by zero.
*   **Decimal Precision:** Supports floating-point numbers while preventing duplicate decimals.
*   **Keyboard Support:** Fully controllable using physical computer keys (numbers, operators, backspace, and enter).
*   **Input Correction:** Includes a "Delete/Backspace" button to erase the last character entered.

## Technologies Used

*   HTML5
*   CSS3
*   Vanilla JavaScript (ES6+)

##  What I Learned

Through this project, I significantly improved my understanding of:
*   Managing complex state and variable flows in JavaScript.
*   Handling string-to-number conversions using `parseFloat()`.
*   Listening to global keyboard events (`window.addEventListener("keydown")`) and preventing default browser behaviors.
*   Structuring clean, modular conditional logic.
